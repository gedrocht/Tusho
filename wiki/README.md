# Tusho Wiki

This directory contains a separate documentation layer powered by DokuWiki.

## Start the wiki locally

```bash
cd wiki
docker compose up --build
```

Then open [http://127.0.0.1:8088](http://127.0.0.1:8088).

## What the wiki is for

The Markdown documentation in `docs/` is structured and concise. The wiki is intentionally more tutorial-like and repetitive so that a complete beginner can build intuition before reading the code.
