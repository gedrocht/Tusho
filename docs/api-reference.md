# API Reference

The GitHub Pages build publishes Doxygen-generated C++ API reference under the `api/` path.

## What you will find there

- public class and struct documentation
- method signatures
- header relationships
- source file cross references

## How to use it well

If you are new to the project, read the tutorials and architecture docs first. The API reference is most useful after you already understand the high-level story of the crawler.
