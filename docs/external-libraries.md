# External Libraries

Tusho deliberately keeps its dependency list small, but you should still know where to read official documentation for the tools it depends on.

## C++ and build tools

- SQLite C library: [https://www.sqlite.org/docs.html](https://www.sqlite.org/docs.html)
- CMake: [https://cmake.org/cmake/help/latest/](https://cmake.org/cmake/help/latest/)
- CTest: [https://cmake.org/cmake/help/latest/manual/ctest.1.html](https://cmake.org/cmake/help/latest/manual/ctest.1.html)

## PHP tools

- PDO SQLite: [https://www.php.net/manual/en/ref.pdo-sqlite.php](https://www.php.net/manual/en/ref.pdo-sqlite.php)
- PHPUnit: [https://phpunit.de/documentation.html](https://phpunit.de/documentation.html)
- PHPStan: [https://phpstan.org/user-guide/getting-started](https://phpstan.org/user-guide/getting-started)
- PHP_CodeSniffer: [https://github.com/squizlabs/PHP_CodeSniffer/wiki](https://github.com/squizlabs/PHP_CodeSniffer/wiki)

## Documentation tools

- Doxygen: [https://www.doxygen.nl/manual/index.html](https://www.doxygen.nl/manual/index.html)
- Material for MkDocs: [https://squidfunk.github.io/mkdocs-material/](https://squidfunk.github.io/mkdocs-material/)
- DokuWiki: [https://www.dokuwiki.org/manual](https://www.dokuwiki.org/manual)
