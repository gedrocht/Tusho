# Wiki Layer

Tusho includes a separate documentation layer based on actual wiki software, not just Markdown pages pretending to be a wiki.

## What is included

- A DokuWiki Docker image definition
- A Docker Compose file to run it locally
- Seed beginner pages that explain the project slowly and carefully

## Why a separate wiki exists

The main documentation site is organized and reference-oriented. The wiki is intentionally slower and more tutorial-like so a new learner can build intuition before reading the lower-level API documentation.
